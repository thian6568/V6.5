# System Rules

1. One artwork upload creates one main artwork record.
2. The same artwork record is reused by marketplace, dashboards, certificates, auth pages, and VR gallery.
3. No duplicate upload path for marketplace and VR gallery.
4. Unity reads only approved artwork data from backend.
5. Marketplace and VR gallery remain synchronized after edits.
6. User authentication and artwork authentication are separate systems.
7. Escrow, subscriptions, shipping, and insurance are compulsory modules.
8. Admin approval is mandatory before public listing visibility.
