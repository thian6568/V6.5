# Artist In Art

Monorepo scaffold for the Artist In Art platform.

## Core rule
One uploaded artwork is stored once in the core backend and reused across marketplace, dashboards, certificates, and WebGL/VR gallery.

## Primary folders
- `apps/web-marketplace`
- `apps/unity-webgl-gallery`
- `backend/supabase`
- `packages/*`
- `docs/*`
- `infra/*`
