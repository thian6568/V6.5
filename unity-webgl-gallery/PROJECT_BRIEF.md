# Artist In Art — Project Brief (unity-webgl-gallery)

## Purpose
`unity-webgl-gallery` delivers immersive gallery experiences in browser via WebGL.

## Stack
- Unity (LTS) + C#
- WebGL build target
- Addressables for environment/content loading
- Backend API integration for approved artwork metadata

## Scope
- Environment metadata and navigation logic
- Wall anchors and artwork placement logic
- Runtime loading of approved artworks from backend
- Visibility synchronization with marketplace rules
