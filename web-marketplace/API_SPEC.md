# API Spec (web-marketplace integration)

- `GET /listings` for browsing
- `GET /listings/:id` for detail pages
- `POST /artworks` upload metadata creation
- `POST /artworks/:id/submit-for-approval`
- `POST /checkout/session`
- `GET /orders/:id`
- `POST /refunds`
- `GET /profiles/me`
- `GET /artworks/:id/analytics`
