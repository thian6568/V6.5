# Requirements (web-marketplace)

1. Login/registration and role-aware routing.
2. Artist dashboard for managing artwork and listings.
3. Admin approval interface for authenticity/listing review.
4. Marketplace browse + search + recommendation UI.
5. Cart + checkout with escrow status UI.
6. Subscription management pages.
7. Refund request flow and order tracking.
8. Social interactions: likes, comments, follows.
9. KYC status prompts and onboarding gating where required.
