# Artist In Art — Project Brief (web-marketplace)

## Purpose
`web-marketplace` provides customer, artist, and admin web experiences.

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS + component library
- Supabase client SDK

## Scope
- Landing page and discovery experiences
- Artist dashboard + upload + analytics
- Admin dashboard + approvals + moderation
- Marketplace, cart, checkout, subscription, refund request pages
